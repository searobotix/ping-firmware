# Ping Firmware
Firmware of [P30 Sensor](http://searobotix.com/p30-sonar/)

<br/>

## P30 通信协议

UART 串口通信 （ TTL 逻辑电平：3.3-5.0 V）

<br/>
	
## P30 线序定义

颜色|定义  
:-:|:-:
白|Tx
绿|Rx
红|VCC
黑|GND  

<br/>

## P30 串行通信波特率？

出厂默认为 115200 。波特率取决于使用设备编程的[固件](https://github.com/searobotix/ping-firmware/tree/master/ping1d)，可改为 9600 。
